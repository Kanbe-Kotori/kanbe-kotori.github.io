近期，随着对新冠疫情的严格管控突然一百八十度大转弯，疫情开始以极其迅速的态势在全国蔓延。尽管作为一个普通民众，对于国家的选择无权置喙，但这同时也意味着国家将是否应该继续防疫的选择权交到了每一个人手上，每一个人在一定程度上仍然保有自己的选择权。本人虽然不是什么传染病学专家，但通过参考一些权威资料，还是总结出了一份自用的防疫自救指南。在这里分享给大家，也希望能够帮到那些决定继续防疫的人。

## 出门防护
首先需要记住新冠的主要传播途径是呼吸道和眼睛，而不是说环境里有病毒就一定会中招。因此保护住这两条路径是重中之重。
+ 出门一定不要摘口罩。
  + 摘口罩时要注意，假装口罩面上全是粑粑，不要用手碰到粑粑。 
+ 任何情况下假设自己的手是污染的。需要用手进行危险操作之前先消毒，洗手或者喷酒精都可以。
  + 消毒前尤其注意不要用脏手碰脸。
其他的防护只是为了进一步降低风险，重要性上都可以往后排。如果连这两条都做不到的话，那剩余的也不用干了。

## 居家防护
被阳性邻居传染的主要途径是下水道和气溶胶，下面我分别从这两个方面提供解决方案。
### 下水道防护
+ 首先，检查所有下水道的水封，即你家下水道与主排水管之间有无U型管连接。水封高度为5cm以上最佳。
  + 如果能闻到臭味，那么肯定没有水封。
  + 每一段时间定时加水防止水封干掉。
+ 可以用装满水的保鲜袋盖住地漏形成双重水封。
+ 马桶冲的时候没有水封，因此冲之前需要盖上盖子。
+ 洗手池需要注意用胶带封住溢流口。

### 气溶胶防护
+ 一种可行的解决方案是crbox，一种3M公司发明的空气过滤装置，由四片任意级别的过滤网和一个任意吸力的排风扇组成。
  + 虽然病毒粒子较小会直接通过滤网，但病毒的载体通常是气溶胶和灰尘，它们会被吸附在滤网上直至病毒自行死亡。
  + 自行组装教程可以参考[链接](https://www.bilibili.com/video/BV1HR4y1f7QU) ，滤网用f8或者h12级别的滤网均可。
  + 在没有疫情的时候也能当个新风用。
  + 品牌空气净化器也能达到类似效果且噪音较小，但相对地，换气量仅能达到crbox的零头。

## 不一定靠谱的邪门招
这是一篇12月5日刚刚刊发在权威杂志nature上的文章，[链接](https://www.nature.com/articles/s41586-022-05594-0) 
+ 文章表明，UDCA可能对预防新冠有效，因为它能关闭人的ace2受体。
+ UDCA即熊去氧胆酸，是一款治疗胆结石的老药，已经面世几十年。
  + 由于这是款老药，副作用相对明确，为2%-9%的腹泻和其余少于万分之一的偶发症状。
  + 在国内的商品名是优思弗，为处方药，具体怎么开还请各位自行想办法。
+ 文章给出的推荐剂量为15mg/kg/d。
+ 肝肾功能不全者禁用。
+ 本条仅供参考，如出现意外本人概不负责。
