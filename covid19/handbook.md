## 前言
近期，随着对新冠疫情的严格管控突然一百八十度大转弯，疫情开始以极其迅速的态势在全国蔓延。尽管作为一个普通民众，对于国家的选择我无权置喙，但这同时也意味着国家将是否应该继续防疫的选择权交到了每一个人手上。本人虽然不是什么传染病学专家，但通过参考一些权威资料，还是总结出了一份自用的防疫自救指南。在这里分享给大家，也希望能够帮到那些决定继续防疫的人。**为了防止有人尬黑我在恰烂钱，本文中不会出现任何商品链接。**

## 为什么要继续防护？
+ 以下观点仅为本人看法。
  + 如果对下面的观点有疑问，欢迎发论文和我对线，也可以直接跳过本节内容。
  + 如果你觉得新冠只是“大号感冒”，那你还来看这篇文章干什么？


+ 关于死亡率
  + 目前Omicron的死亡率约为千分之一到千分之二。
  + 新冠是一种变异速度极快的RNA病毒，很难预知下一个变种的致死率。
  + 不存在“病毒随着传播致死率降低”的说法，中学教科书上那个广为流传的兔粘液瘤病毒的例子，后续见[链接](https://www.pnas.org/doi/abs/10.1073/pnas.1710336114)。


+ 关于是否存在后遗症或Long Covid
  + 目前科学界没有得出统一结论，主要原因是新冠诞生时间尚短，而长期后遗症的时间跨度可不止这点。
  + 从其致病机理来看，本人倾向于认为长期后遗症很少见。
  + 但新冠**在短期内肯定会造成各种症状**，这点也是不可否认的，其中包括但不限于免疫功能下降（尤其是细胞毒T细胞的减少）、运动耐量下降、脑雾、凝血功能异常、内分泌失调（尤其是糖尿病）、味觉/嗅觉失常等。
  + 然而相比担心长期后遗症，**不如花点心思多想想如何避免反复感染**，因为在免疫功能下降期间感染其他病原体，或再次感染新冠，很可能造成严重后果，甚至是致命的。参见[链接](https://www.nature.com/articles/s41591-022-02051-3)。


+ 关于反复感染
  + 可以参考[近期信息汇总](recent.md)


+ 已经做好防护，但还是不幸被破防？
  + 别担心，你做的防护能有效减少进入身体的病毒数量。有证据表明，首次进入身体的病毒数量越少，出现的症状就越轻，后遗症也越少。


## 外出防护

首先需要记住新冠的主要传播途径是**呼吸道和眼睛，而不是说环境里有病毒就一定会中招**。因此保护住这两条路径是重中之重。

+ 出门一定要佩戴口罩，在任何情况下尽量不要摘口罩。
  + 检查口罩气密性的方法：双手盖住口罩，感觉**在呼气的时候口罩是否有向外轻微的膨胀**。
  + 进入日本厚劳省总结的“三密”场所，即**密闭、密集、密切接触**场所时，最好佩戴N95口罩。
  + 需要更换口罩时，寻找一个**通风的无人场所**，然后**对手进行消毒**后更换，然后丢弃旧口罩并再次消毒双手。
  + 其他的防护只是为了进一步降低风险，重要性上都可以往后排。**如果连这一条都做不到的话，那其他的也不用干了。**


+ 除呼吸道外，**眼睛亦是传染的重灾区**，因此面屏/护目镜有助于进一步降低感染风险。
 

+ 建立基本的洁净与污染观念。
  + 洁净区即绝对不可能存在活病毒，而污染区则存在一定风险。
  + 用酒精消毒过的手和东西是洁净的，口罩内侧和衣兜是洁净的。
  + 外衣外裤是污染的，口罩外侧是污染的。
  + **一旦洁净区和污染区发生接触，则洁净区立刻变成污染的。**


+ 一些具体的操作举例
  + 假装任何处于污染区的东西表面沾满了屎，不要用手触碰到屎。碰到屎后立即洗手或消毒。
  + **任何情况下假设自己的手是污染的**。需要用手进行危险操作之前先消毒，洗手或者喷酒精都可以。
  + 摘口罩时注意不要用手碰到口罩面。 
  + 尤其注意**不要用脏手碰脸**。
  + 如果对酒精或消毒液过敏或不耐受，**一次性手套**可能是个好选择。


## 居家防护

+ 家中有条件的话最好能分出洁净区和半污染区。
  + 回家后，把最外层的衣服挂在半污染区，之后不要再碰这一区域。鞋底也要注意消毒，消毒地垫是一个比较方便的选择。
  + 消毒双手后方可摘下口罩，然后立即洗手。
  + 其他物品如快递的消杀也在此区域完成。
  + 鉴于有些东西不方便用酒精或含氯消毒液消毒，在此区域放置紫外线灯可能会有帮助。


+ 关于小区传播
  + 被阳性邻居传染的主要途径是**下水道**和**气溶胶**，下面本文将分别从这两个方面提供解决方案。

### 下水道防护
+ 首先，检查所有下水道的水封，即你家下水道与主排水管之间有无U型管连接。水封高度为5cm以上最佳。
  + 如果在家总能闻到下水道反的臭味，那么肯定没有水封。
  + 对于不常用的地漏，每一段时间需要定时加水防止水封干掉。


+ 可以用装满水的保鲜袋盖住地漏形成双重水封。


+ 马桶冲的时候没有水封，因此冲之前需要盖上盖子。


+ 洗手池需要注意用胶带封住溢流口。

### 气溶胶防护
+ 一种可行的解决方案是crbox，3M公司发明的空气过滤装置，由四片任意级别的过滤网和一个任意吸力的排风扇组成。
  + 虽然病毒粒子较小会直接通过滤网，但病毒的载体通常是气溶胶和灰尘，它们会被吸附在滤网上直至病毒自行死亡。
  + 自行组装教程可以参考[链接](https://www.bilibili.com/video/BV1HR4y1f7QU) ，滤网用f8或者h12级别的滤网均可。
  + 在没有疫情的时候也能当个新风用。


+ 成品的空气净化器也能达到类似效果。不仅噪音较小，而且适合手残。但相对地，换气量相比crbox有所不足，性价比也远低于前者。选择哪种见仁见智。


+ 主动放出羟自由基的空气消毒机（其实很多除味器也是同样原理）可以更主动地消杀环境里的病毒，但是否对身体造成伤害目前还存疑。实在要用的话最好还是戴上口罩，减少自由基直接和呼吸道反应。

## 药品准备

虽然本文上面的部分全都在讲如何防护，但在国内普遍高人口密度的大背景下，想百分之百防止感染也确实不太现实。鉴于第一波疫情来袭时医院的挤兑情况，在家少量备药肯定是必须的。**就算现在一时买不到，在下一个高峰来临前也最好备齐。**

+ 首先，退烧药肯定是必备的。
  + **扑热息痛/布洛芬/双氯芬酸钠**，以及任意含有上述成分的感冒药均可。
  + 含有**金刚烷胺**或**奥司他韦**的感冒药**不推荐**，那是治流感的。
  + 关于扑热息痛，这里需要额外多说几句。**这个药是好药没错，但不能多吃。**多吃的扑热息痛无法以无毒形式被代谢，而会被氧化成有毒的N-乙酰苯醌亚胺，在耗竭肝脏的解毒剂还原性谷胱甘肽之后，造成严重的肝损伤。因此，请**严格按照剂量服用**。此外，不少复方感冒药中都含有这一成分，因此**不要一下吃好几种**。


+ 关于化痰药
  + 最好准备一些。
  + 氨溴索（沐舒坦）和溴己新药理作用相同，不需要重复准备。
  + 乙酰半胱氨酸和羧甲司坦也可。


+ 关于止咳药
  + 如果你的复方感冒药中含有右美沙芬，无需准备。
  + 否则，推荐含有可待因成分的止咳药，但不能长期服用，也不能和柚子一起服用。
  + 其他中成药如苏黄止咳也可。


+ 关于抗生素
  + 我个人建议是备一点，**主要是怕到时候被抢空。**
  + 为什么病毒感染要备抗生素？因为身边很多人退烧后又复发低烧甚至白肺，经检查发现是免疫下降导致的细菌感染。
  + 经验性用药：左氧氟沙星+头孢，这些药都是处方药，请遵医嘱使用。
  + **该条不构成用药建议，本人也不负责任。**
  + 如果还不退烧，直接送医院。


+ 关于止泻药和补液盐
  + 不知道为什么坊间突然流传起新毒株会导致腹泻，导致**蒙脱石散**脱销，就很怪。。
  + **补液盐**的作用是**防止大量腹泻后电解质紊乱**，作用约等于**电解质饮料**。
  + 如果能买到的话就备一点吧，**没有也无所谓**，拉两天稀就拉吧。


+ 关于辉瑞神药
  + 副作用很大，甚至堪比化疗，因此谨慎使用。
  + 仅对高危人群防止转重症有效，对非高危人群弊大于利。
  + 家里有老年人有基础病，且家里有矿的话可以备一点，其他人不推荐。
  + 怎么开？家里有矿的人自有路子，我没有，所以我也没备。


+ 关于阿兹夫定
  + 是治艾滋病的老药，对治疗新冠可能有用。
  + 该药遗传毒性阳性，有潜在的致癌性。
  + 该药生殖毒性也为阳性，有潜在的致畸性。
  + 利弊请自己权衡。


+ 关于默沙东的Molnupiravir
  + 其原理是增加病毒RNA复制时出现的错误，导致病毒在复制过程中自灭。
  + 可能会诱导病毒产生危险突变...
  + 同样，利弊请自己权衡。


+ 关于熊去氧胆酸（UDCA）
  + 出处：12月5日发表nature上的一篇文章，[链接](https://www.nature.com/articles/s41586-022-05594-0) 
  + 由于能关闭人的ace2受体，因此可能对预防新冠有效，但治疗效果较差。
  + 推荐剂量请点击原文获取，看不懂原文的人不建议使用。
  + 肝肾功能不全者禁用。
  + **该条不构成用药建议，本人也不负责任。**
